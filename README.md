# Sequential Sampling Models

Collection of computational models for decision-making based on sequential sampling frameworks.

## Models included

* **Diffusion Decision Model (DDM)**<sup>[1]</sup>
* **Gated Cascade Diffusion Model (GCDM)**<sup>[2]</sup>
* **Leaky Integrating Threshold (LIT)**<sup>[3]</sup>

All models are implemented in **C** and **Python**.

Each implementation provides explicit equations, parameter definitions (with units) and usage examples.

[3] Verdonck, S., Loossens, T., & Philiastides, M. G. (2021). *The Leaky Integrating Threshold and its impact on evidence accumulation models of choice response time (RT)*. *Psychological Review*, 128(2), 203–221. https://doi.org/10.1037/rev0000258
